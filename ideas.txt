# В игре есть карта, с несколькими локациями: дом, спортзал, магазин, автодилер, гараж, заправка, работа, банк.

### Прокачка навыков - в спортзале ###
# Прокачка навыков находится в спортзале. Навыки можно прокачивать за шаги: 1к/2к/3/4к/5к/...+ за каждый навык.
# Каждая эдиница навыка - это коэффициент. Изначально коэффициент 1.00.
# Каждая эдиница прокачки навыка дает + 0.01 в коэффициенту.
# Так же коэффициент можно увеличивать за счёт вещей в инвентаре, найденых артефактов в приключениях,
# за счёт погоды, и в разных приключениях будут разные коэффициенты.

##### Характеристики персонажа #####
# - Выносливость. Добаляет + 0.01 % пройденых шагов за день.
# - Энергия. Доблавяет + 5 энергии.

# - Механика. Добавляет +1 к навыку, +1 % меньшему износу вещей, +1 % шанса на ремонт вещи, +10% ЗП
# Механика нужна для открытия определенных механических вещей в игре. Возможности чинить транспорт и технику.
# - IT Технологии. Добавляет +1 навыку, 1% меньшему износу электронных вещей, +1% шанса на ремонт электройнной вещи, +10% ЗП
# IT Технологии нужны для возможности удаленной работы. Возможности использовать различные электронные устройства

# - Сопротивляемость к холоду
# - Сопротивляемость к жаре

# - Навык ходьбы по горной местности
# - Навык ходьбы по равнинной местности
# - Навык ходьбы по лесу
# - Навык ходьбы по болоту
# - Навык ходьбы по снегу
# - Навык ходьбы по льду
# - Навык ходьбы по песку
# - Навык ходьбы по камням
# - Навык лазить по горам и камням (Нужен для преодоления определенной месности)

# - Навык экономии сил в походе. +0.01% кол-ву пройденых шагов в путешествии
# - Навык логистики в походе. -0.01% общей длины маршрута. -1% вероятности заблудится на маршруте.


# - Удача. Добавляет % к выпадению события. Так же добавляет % к удачному результату
# - Кулинария. Добавляет возможность готовить различную еду как дома, так и в походе.
# - Рыбалка. Возможность ловить рыбу в походе и на озере в центральной локации.
# - Экономная езда на транспорте. +1 ед. -1% расхода топлива
# - Спокойная езда на транспорте. +1 ед. -1% расхода енергии на поездку
# - Логистика при езде между локациями -1% вероятности заблудится по дороге на транспорте

### Магазин ###
# В магазине можно покупать разное оборудовае и инвентарь для путешествий.
# Покупка еды, для потушествий, и что бы восстанавливать запас эгергии
# Экипировка: обувь, штаны, куртка, перчатки, рюкзак разного объема

### Банк ###
# В банке можно обменять шаги на золотую валюту.
# Возможно, брать кредит на путешествие и на покупки.

### Приключения ###
# Приключения могут быть как короткими, так и длинными на несколько дней. 1-3, 3-7.

### Транспорт ###
# В зависимости от транспорта, будет тратится различное кол-во шагов и энергии, для перехода между локациями
# Затраты на перемещение между локациями:
# Шаги:
# Пешком: 500, Велосипед: 250, Скутер: 100, Кемпер: 150, Машина: 50.
# Энергия:
# Пешком: 20, Велосипед: 10, Скутер: 5, Кемпер: 10, Машина: 2
# Транспорт можно купить у дилера: велосипед, скутер, машина, кемпер.
# Для разных приключений нужен будет свой транспорт
# На кемпере можно будет поехать в особый тип приключения, на несколько дней
# Самолет для перелета в другой регион
# На транспорт можно вешать различные багажники и боксы на крышу.
